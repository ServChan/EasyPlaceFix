# EasyPlaceFix
Litematica Easy Place Fix — это небольшой клиентский мод для Fabric, дополняющий Litematica и исправляющий ориентацию блоков при использовании режима Easy Place на мультиплеерных серверах. // Litematica Easy Place Fix is a small Fabric client-side addon for Litematica that corrects block orientation when using Easy Place mode on multiplayer servers
